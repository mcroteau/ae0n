PLSA.R is an Open Source Java Server + Framework for high demand, low latency requirements. There are no static references, no file reads, access to static fields per request. Everything is either cached and or instantiated on the fly. PLSAR runs via one command so there are no .war files to deploy, no additional plugins to install it is a simple yet sophisticated alternative to container deployment environments.
