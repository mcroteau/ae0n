PLSA.R
www.plsar.net

PLSA.R is an Java Server + API for high demand, low latency requirements. PLSA.R runs via one command so there are no .war files to deploy, no additional plugins to install it is a simple yet sophisticated alternative to container deployment environments. Performance wise, in theory, it can handle any type of concurrent request situation as long as your computer can handle the load.
